[JD] 软件工程师招聘

算法工程师

    职位描述

  • 根据用户的需求，完成算法设计、实现和调优，推进算法落地
  • 负责指纹比对、图像搜索、深度学习算法等领域的研发工作
  • 追踪技术前沿，判断最新技术对产品的影响

    职位要求

  • 有优秀的分析问题和解决问题的能力，乐于挑战困难的研究课题
  • 熟悉 Linux 操作系统，能熟练使用 C++ 和 Python 开发
  • 熟悉深度学习技术，熟悉至少一个深度学习框架（MXNet/TensorFlow/PyTorch/Caffe）
  • 对机器学习、图像搜索、指纹比对等领域有强烈好奇心

后端系统工程师

    职位描述

  • 负责产品（如指纹比对和图像搜索）相关的后端架构设计和研发工作
  • 完成所负责模块的设计文档编写、架构设计、编码、测试的全流程工作
  • 持续优化后台系统架构，提高系统在高并发、分布式环境下的容错能力，保证系统的可运维、高可用性、高可靠性
  • 培训和协助运维人员进行部署和维护

    职位要求

  • 软件开发知识全面，基础扎实，精通常用数据结构与算法
  • 熟悉 Linux 操作系统，熟悉 shell，能熟练使用一种后端语言（如 C++/Java/Go/Rust，目前后台主要使用 Java）
  • 熟悉常见分布式系统，例如分布式消息队列、分布式文件系统等系统架构，并能合理应用分布式技术，解决问题
  • 熟悉数据库体系结构，熟悉常见的索引、存储架构和技术，对 RDBS 和 NoSQL 有实际开发、使用经验
  • 有钻研精神，学习能力强，有责任心和团队合作精神

前端全栈工程师

    职位描述

  • 开发应用人工智能技术的企业级应用，包括前端网站以及后端业务层开发
  • 公司内部的调试工具和管理工具开发
  • 国内外大型科技展会的 demo 开发

    职位要求

  • 扎实的数据结构和算法知识
  • 熟练掌握至少一种现代前端框架（如 React/Vue/Angular/Cycle 等）
  • 熟练掌握一门后端语言（如 Python/Java/Go/Ruby 等）和一种后端开发框架（如 Sprint Boot/Ruby on Rails/Gin 等）
  • 熟悉一种 SQL/NoSQL 数据库（如 MySQL/PostgreSQL/MongoDB 等）
